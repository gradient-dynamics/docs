# Keyboard Shortcuts

Quick reference for keyboard shortcuts in the Gradient Dynamics Studio workspace.

## 3D Viewer

| Shortcut | Action |
|----------|--------|
| **Left-click + drag** | Rotate the view |
| **Right-click + drag** | Pan the view |
| **Scroll wheel** | Zoom in/out |
| **Shift + left-click** | Pan the view (alternative) |
| **Middle-click + drag** | Pan the view (alternative) |
| **Double-click** | Center view on clicked point |

## Camera Presets

| Shortcut | View |
|----------|------|
| **1** | Front |
| **2** | Back |
| **3** | Left |
| **4** | Right |
| **5** | Top |
| **6** | Bottom |
| **0** | Isometric |
| **F** | Fit all (zoom to show entire geometry) |

## Selection

| Shortcut | Action |
|----------|--------|
| **Click** | Select a surface/face |
| **Shift + click** | Add to selection |
| **Escape** | Clear selection |

## Display

| Shortcut | Action |
|----------|--------|
| **W** | Toggle wireframe overlay |
| **T** | Toggle transparency |
